📥 从微信群采集消息

项目通过 chatlog 读取本机微信数据库并暴露 HTTP API， 再由 scripts/collect_from_chatlog.py 拉取指定群并调用后端 POST /api/messages/import 入库。 

# 1) 启动 chatlog HTTP API（默认 127.0.0.1:5030）
chatlog server

# 2) 首次全量同步某个群（--reset 会清掉上次断点）
export PORTAL_ADMIN_TOKEN=你的Token
python3 scripts/collect_from_chatlog.py \
    --chat "群名称或chatroom_id" --reset --once

# 3) 常驻增量轮询（每 60 秒拉一次新消息）
python3 scripts/collect_from_chatlog.py \
    --chat "群名称或chatroom_id" --interval 60

# 4) 想加第二个群？把 --chat 换成另一个名字再跑一次即可
#    入库后到本页顶部「选择要统计的群」切换即可

每个群单独一个采集进程；入库后自动出现在上方群列表，选中即可切换门户展示。

📖 使用流程

  1. 先配置 AI — 选择服务并填 API Key。
  2. 再采集消息 — 用 scripts/collect_from_chatlog.py 从微信群持续读取。
  3. 生成每日精华 — 在每日精华页面点击 重新生成，AI 会输出结构化 JSON。
  4. 抽取同营人自我介绍 — 名册每张卡片右下角点一次即可。
  5. 部署到服务器 — 参考 deploy/ 目录里的 systemd + nginx 示例。
